Admin Guide

Complete guide to managing your ShopHub e-commerce platform.

Create Admin User

Step 1: Run the Admin Creation Script
npm run create-admin
Default Admin Credentials
  • Email: admin@shophub.com
  • Password: admin123
Important: Change the default password immediately after first login!
Creating Additional Admins

To create additional admin users, you can either:

  1. Register a normal user account
  2. Update their role in MongoDB to 'admin'
// Using MongoDB shell
db.users.updateOne(
  { email: "newadmin@example.com" },
  { $set: { role: "admin" } }
)

Access Admin Panel

Login

Navigate to the login page and enter admin credentials

http://localhost:3000/login.html
Access Panel

Click "Admin Panel" link or navigate directly

http://localhost:3000/admin.html
The "Admin Panel" link appears in the navigation bar only for users with admin role.

Security Best Practices

Critical Security Measures
  • Change Default Password: Immediately after first login
  • Use Strong Passwords: Minimum 12 characters with mixed case, numbers, and symbols
  • Logout After Use: Always logout when finished
  • Regular Backups: Backup database regularly
  • Monitor Activity: Check for unusual orders or products
Access Control
  • Limit admin access to trusted personnel only
  • Review admin user list regularly
  • Remove inactive admin accounts
  • Use separate accounts for different admins

Troubleshooting

Solutions:
  • Ensure you're logged in with admin credentials
  • Check user role is set to 'admin' in database
  • Clear browser cache and cookies
  • Verify JWT token is valid
