Getting Started

Follow this guide to set up and run ShopHub on your local machine or server.

1Prerequisites

Before you begin, ensure you have the following installed:

Node.js

Version 14.x or higher

Download
MongoDB

Version 4.x or higher

Download
Tip: You can also use MongoDB Atlas (cloud database) instead of local MongoDB.

2Installation

Clone the Repository
git clone https://github.com/yourusername/shophub.git
cd shophub
Install Dependencies
npm install
Note: This may take a few minutes depending on your internet connection.

3Configuration

Create Environment File
cp .env.example .env
Edit .env File

Open the .env file and configure the following variables:

# Server Configuration
PORT=3000
NODE_ENV=development

# Database
MONGODB_URI=mongodb://localhost:27017/ecommerce

# JWT Secret (Change this!)
JWT_SECRET=your_super_secret_jwt_key_change_this_in_production

# M-Pesa Daraja API (Optional)
MPESA_ENVIRONMENT=sandbox
MPESA_CONSUMER_KEY=your_consumer_key
MPESA_CONSUMER_SECRET=your_consumer_secret
MPESA_SHORTCODE=174379
MPESA_PASSKEY=your_passkey
MPESA_CALLBACK_URL=https://yourdomain.com/api/payment/mpesa/callback

# PayHero API (Optional)
PAYHERO_API_KEY=your_base64_encoded_api_key
PAYHERO_CHANNEL_ID=911
PAYHERO_CALLBACK_URL=https://yourdomain.com/api/payhero/callback
Security: Never commit your .env file to version control!

4Database Setup

Start MongoDB

Make sure MongoDB is running on your system:

# On Linux/Mac
sudo systemctl start mongod

# Or using MongoDB service
mongod
Seed Sample Data (Optional)

Populate your database with sample products:

npm run seed
Success: Sample products have been added to your database!

5Create Admin User

Create an administrator account to access the admin panel:

npm run create-admin
Default Admin Credentials:
  • Email: admin@shophub.com
  • Password: admin123
Important: Change the admin password immediately after first login!

6Running the Application

Development Mode

Start the server with auto-reload on file changes:

npm run dev
Production Mode

Start the server in production mode:

npm start
Server is running at http://localhost:3000
Access Points
Page URL Description

Products http://localhost:3000/products.html Product catalog

Documentation http://localhost:3000/docs/ This documentation

7Testing

Test User Registration
  1. Navigate to http://localhost:3000/register.html
  2. Create a new user account
  3. Login with your credentials
Test Product Browsing
  1. Go to Products page
  2. Browse available products
  3. Add items to cart
Test Admin Panel
  1. Login with admin credentials
  2. Navigate to http://localhost:3000/admin.html
  3. Try adding/editing products
